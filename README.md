
# Student Resource Portal (UMSI CDO) — Client Project

A responsive, mobile-first, accessible website concept/design demo for the UMSI Career Development Office (CDO) to help students find career resources (exploration, resumes, networking, job search).

**Live:** https://heycoming.github.io/UMSICDOResourcePortal/  

---

## Pages
- `index.html` — Home / key resources
- `resume.html` — Application materials (resume/portfolio)
- `netCon.html` — Networking & connections
- `plan.html` — Career exploration / planning
- `404.html` — Custom 404 page with navigation back to main sections

---

## Key Features
- Mobile-first responsive design with breakpoints at **768px**, **1024px**, **1280px**
- UM-inspired **blue/yellow** styling
- Accessibility: **semantic headings**, **alt text**, **skip-to-main link**
- Navigation includes a styled **active/current page**
- Enhanced footer (contact + quick links + social links)
- Custom 404 page with a friendly design/animation
